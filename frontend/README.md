# Frontend

Статический сайт IDM: HTML, CSS, JavaScript.

## Локальный просмотр

```bash
npx --yes serve frontend -p 8080
```

Откройте http://localhost:8080

## Структура

| Путь | Назначение |
|------|------------|
| `*.html` | Страницы сайта |
| `css/`, `styles/` | Стили |
| `js/` | Скрипты (формы, аналитика, API) |
| `img/` | Изображения |

Настройки: `js/site-config.js`. Аналитика: [ANALYTICS.md](../ANALYTICS.md).

## GitHub Pages

В настройках репозитория: **Pages** → Branch `main` → папка **`/frontend`**.
